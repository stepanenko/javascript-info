# Object

Creating Objects:

- Object Literals:
```js
const person = { name: "Alice", age: 30 };
```
- Constructor Function:
```js
const person = new Object();
person.name = "Alice";
```
- Classes (ES6): Using class syntax
```js
class Person {
  constructor(name, age) {
    this.name = name;
    this.age = age;
  }
}
```
